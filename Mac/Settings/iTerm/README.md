# Setting Keyboard Keys to Be Mac Style

These keys do not work out of the box for `iterm`

| Key Combos       |
|------------------|
| Opt+Backspace    |
| CMD+Backspace    |
| Opt+Left Arrow   |
| CMD+Left Arrow   |
| Opt+Right Arrow  |
| CMD+Right Arrow  |
| Opt+Up Arrow     |
| CMD+Up Arrow     |
| Opt+Down Arrow   |
| CMD+Down Arrow   |


- Go to iTerm Preferences 
- Go to Profiles
- Select your profile
- Go to the `Keys` tab
- Click Load Preset... 
- Choose Natural Text Editing.
